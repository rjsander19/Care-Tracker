# Care-Tracker

A web application that allows the user to create and maintain task lists for different aspects of their life (pet care, household, work, acedemics, etc.)

## Types of Technology used:
- HTML 5
- CSS 3
- JavaScript
- JQuery
- MongoDB
- Heroku
- Express
